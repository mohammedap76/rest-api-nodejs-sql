A basic REST API built using **Node.js**, **Express**, and **PostgreSQL**. Includes simple user routes.

- GET users endpoint
- PostgreSQL connection with `pg`
- Modular folder structure

## 🔧 Tech Stack
- Node.js
- Express.js
- PostgreSQL
- dotenv
